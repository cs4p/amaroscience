# AmaroScience Blog - Simplified Static Site

Clean, simplified static version of the AmaroScience blog.

## Structure

- **index.html** - Main homepage
- **index-simple.html** - Alternative simplified homepage  
- **style.css** - All styles consolidated in one file
- **posts/** - Blog posts with clean URLs
- **archives/** - Archive pages by date

## Features

✅ Simplified: All CSS in one file
✅ Clean URLs: Readable file names
✅ Flat structure: Easy to navigate
✅ Minimal dependencies: Only Google Fonts
✅ Fast: No unnecessary scripts
✅ Portable: Host anywhere

## Deployment

**Netlify:** Drag and drop folder
**Traditional Host:** Upload via FTP
