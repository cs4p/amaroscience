Georgia law dictates a mandatory runoff if no candidate exceeds 50%.

Given it’s not random why some votes haven’t been counted and given Kemp is very much opposed to having them counted, I think we can assume they are balanced in Abrams favour.  So how much does it need to be balanced in her favour for to spark a runoff? That answer depends how many ballots are outstanding. Here are a few scenarios I quickly whipped up using this very simple [calculator](https://docs.google.com/spreadsheets/d/1YuUngdjgnnqcv4bDQD4LXw3jNs5yrB4UQNmrh3hxJZY/edit?usp=sharing).  This calculator actually slightly favours Kemp in that you only put in the % support Abrams would need to have and it assumes Kemp takes all the remaining support, when in reality, Metz could take some of those votes.  This was to simplify the calculation and since I’m a known liberal conspirator, I’m going to give Kemp this slight edge in the calculation.

As of this writing, Kemp has just shy of 50.3%.

There is a dispute between the number of provisional ballots in existence:

1. Kemp says there are only [20k](https://vancouversun.com/pmn/news-pmn/the-latest-kemp-declares-win-says-rival-using-old-math/wcm/74e7561e-bd50-41a2-82d6-c2b80d794cce) provisional ballots  
2. Abrams team says they collected  [31k](https://www.vox.com/policy-and-politics/2018/11/8/18076562/georgia-abrams-kemp-midterm-election-lawsuit-votes) names from counties of provisional ballots casts.  
3. A judge in Dougherty county has ruled GA [accepts ballots postmarked by election day](https://www.thestate.com/news/nation-world/national/article221416515.html)  
4. Still unclear how many more outstanding ballots there are

Basically as long as Abrams has roughly 60% support on enough of the outstanding ballots, there will be an automatic runoff. I built a quick interactive version of my calculator which I published to [Tableau Public](https://public.tableau.com/profile/talia6974#!/vizhome/Georgia2018ToRunnofforNottoRunoff/Dashboard1?publish=yes).  You can use it to create your own scenarios and see what the outcome would be.  

Here are some probable scenarios that would spark and automatic runoff:

  * If there are at least 150k outstanding ballots, she only needs 57.5% support  
* Even if they only count the roughly 31k ballots team Abrams has found collecting county records, she only needs to exceed 85% support amongst that small group to force an automatic runoff
